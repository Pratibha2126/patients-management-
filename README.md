
 Patients Management System

A simple React-based web application for managing patient details.  
This project demonstrates a basic login system, patient data form, and viewing functionality with a responsive UI.



 Live Demo
https://jazzy-gelato-30bcb7.netlify.app/



 Features Implemented
- Login page (UI only)
- Patient details form with input validation
- View patient data in a table
- Responsive design for desktop and mobile



Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS
- Netlify (for deployment)



Setup Steps
1. Clone the repository:
   bash
   git clone  
   
2. Navigate into the project folder:
   bash
   cd patients-management
   
3. Install dependencies:
   bash
   npm install
   
4. Run the development server:
   bash
   npm run dev
   
5. Open `http://localhost:5173` in your browser (default Vite port)



 Assumptions & Future Improvements
- Authentication is basic (UI-only)
- Currently, patient data is temporary (can integrate backend later)
- Form validation, search/filter, and role-based access can be added with more time
- UI enhancements and error handling improvements
